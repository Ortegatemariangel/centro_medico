# centro_medico_Ortegate

Este proyecto es una base de datos de un centro médico que sirve para organizar la información de pacientes, médicos, citas, usuarios y consultorios.

Permite agendar citas, guardar datos importantes y llevar un control básico del sistema.

### Objetivo

Organizar mejor la atención en un centro médico y facilitar el manejo de la información.
